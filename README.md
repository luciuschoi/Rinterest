# Rinterest

: A sample Rails project to demonstrate the Pinterest-like styled page layout.

## Prerequisite

1. System Postgresql installed on local machine

2. Database creation and migration
  ```sh
  $ rails db:create
  $ rails db:migrate
  ```

3. Data seeding
  ```sh
  $ rails db:seed
  ```

4. Start rails server

   ```sh
   $ rails server
   ```
